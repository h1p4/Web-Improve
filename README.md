📌 Descripción

Este proyecto tiene como objetivo tomar páginas web existentes, obtener una copia local de su estructura y recursos utilizando herramientas como HTTrack, y posteriormente realizar mejoras visuales, de rendimiento y experiencia de usuario para presentar propuestas renovadas a potenciales clientes.

Cada proyecto incluido en este repositorio representa una versión optimizada de una web original, manteniendo la esencia del negocio pero aplicando mejoras modernas de diseño y desarrollo.

🛠️ Tecnologías y Herramientas
HTML5
CSS3 / SCSS
JavaScript
Bootstrap / TailwindCSS
HTTrack Website Copier
Git & GitHub

Clonado local de la web original mediante HTTrack.
Limpieza y organización del código fuente.

Optimización responsive y rendimiento.
Implementación de mejoras modernas:
Diseño responsive
Mejor accesibilidad
Optimización visual

Presentación de la propuesta final al cliente.
🎯 Objetivos
Modernizar páginas web desactualizadas.
Crear propuestas visuales atractivas para clientes.
Mejorar experiencia de usuario y rendimiento.
Construir un portfolio profesional de rediseños web.

⚠️ Aviso Importante

Este repositorio tiene fines educativos, demostrativos y de presentación de propuestas de mejora.

Las páginas originales pertenecen a sus respectivos propietarios.
Las versiones modificadas no se utilizan con fines comerciales sin autorización del propietario original.
